Assignment 6

Using a combination of MySQL and PHP, this assignment requires that a database of our choosing by added to, updated, and have records be deleted through an inteface that we design for the user. In this case, my student database allows the user to enter in relevant information, with error checking for the required fields, and insert it into the database. The page refreshes, allowing the user to see the database on the index page, which is done using a select statement to collect all records in the databse. Any record can be changed or deleted one at a time by whichever one the user selects.

  • SQL: syntax, INSERT, SELECT, UPDATE, DELETE
